# Workshop

A small game prototype with a clear feature roadmap.

## Current focus
Tuning settings are the immediate task. The goal is to make the gameplay feel more customizable and better balanced by exposing core values such as jump height, gravity, obstacle speed, and spawn rate.

## Feature roadmap
- Nice graphics
- Sounds
- Difficulty ramp
- Tuning settings

See [docs/FEATURE_ROADMAP.md](docs/FEATURE_ROADMAP.md) for the full plan.
